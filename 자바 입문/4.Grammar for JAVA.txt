*변수
변수란 값을 저장할 수 있는 메모리의 공간

선언 : 타입 변수이름; ex) int count; double average;
명명(이름정하는)규칙-안 지키면 에러 : 하나 이상의 글자로 이루어짐,첫 번째 글자는 문자이거나 $,_이어야함,두 번째 이후의 글자는 숫자,문자,_,$이어야한다.
                       $,_ 이외의 특수문자 사용 불가,길이제한없음,키워드는 식별자로 사용할 수 없다. 상수값을 표현하는 단어는 식별자로 사용 불가.

식별자는 클래스,메소드,변수 등 다양한 대상에 대해 이름이 붙어지는 경우,그 이름을 뜻함.

관례-안지켜도 상관은없음 : 첫번째 문자가 소문자인 명사로 정한다,여러 단어로 구성된 이름의 경우 두번째 단어부터 첫글자를 대문자로 한다. _를 쓰지 않는다.

* 상수
상수랑 수식에서 변하지 않는 값을 의미한다.

상수의 선언 : final 상수타입 상수명; -> final int J;
상수의 사용  : 상수명 = 값; -? J = 10; //J에 담을수있는 기회가 단 한번만 허락됨
상수의 명명 규칙 : 대문자로만 구성된 명사,여러단어로 구성된 경우 _을 사용하여 구분
상수 사용이유: 값이 변하면 위함한 경우,한눈에 값을 알아볼수있다.

변수의 타입
기본형 : 정수형,실수형,문자형,불린형

논리형 : ture,false의 값만 가질수있다.
문자형 : char로 ''을 사용하여 표현가능
정수형 : 정수를 표현하는 데이터 타입. 주로 int형을 사용-4byte(long형 - 8byte)
실수형 : float,double형이 있다.(4byte/8byte)
리터럴 : 소스 코드의 고정된 값을 의미(ture,false,10,11.1,a 등 이런 값 자체)

기본 타입 (형)변환
변수or리터럴 타입을 다른 타입으로 변환

묵시적 형변환 : int x =50000; long y =x;
명시적 형변환 : 크기가 더 큰 타입을 작은 타입으로 바꿀 때에는 명시적으로 변환해주어야한다. 
                long x =50000; int y = (int)x; // 강제 형변환이라고도 한다.
